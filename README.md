A simple unit converter app that converts between different units of measurement.

---

## Project Overview

**Tech Stack:**
- Frontend: React (Vite)
- Backend: Node.js + Express
- Storage: In-memory (No database)

**Current features:**
- Convert between different units of measurement
- View conversion history
- Add custom conversion units
- Save favorite conversions
- Real-time conversion results

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/devsoc-aec/unit-converter.git
```

### 2. Install dependencies
**Server**
```bash
cd Server
npm install
npm run dev
```

**Client**
```bash
cd Client
npm install
npm run dev
```

### 3. Access the application
- Frontend: http://localhost:5173
- Backend: http://localhost:5000
